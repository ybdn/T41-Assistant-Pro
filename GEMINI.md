# GEMINI

Ce document fournit des informations essentielles pour les agents IA (comme Gemini) interagissant avec ce projet.

## À propos du projet

Cette extension de navigateur analyse les pages web et fournit des diagnostics.

## Conventions du projet

- Le code est écrit en JavaScript.
- Les noms de fichiers et de variables sont en camelCase.
- Les commentaires sont en anglais.

## Commandes clés

- Il n'y a pas de framework de test ou de build system. Les modifications sont testées manuellement.
